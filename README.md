# Price-Comparison-App
Mobile application for comparing product prices across different stores.

## Assignment 1 Scope (UI Foundations)

- Architecture: MVVM (`UI -> ViewModel -> State -> UI`)
- Screens: `Home`, `Products`, `Details`, `Add Product`, `Favorites`
- Compose components extracted per-screen under `ui/screens/<screen>/components`
- BIH-focused sample data and cities, with English UI text
- Validation: add-product form requires valid name, price, category, and store
- Edge cases: empty city product list and empty favorites state
